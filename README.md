# passwordManager
Project PasswordManager
Password manager project built with <b>React + Tailwind</b>.<br>
The project allows you to add the website username and password used on your website and temporarily stores them in <b>localStorage</b>.<br>
It allows you to edit or delete saved passwords.<br>
Saved passwords are hidden and can be viewed by clicking the eye button.<br>
It features a footer with modern icons and links to social networks.<br>
